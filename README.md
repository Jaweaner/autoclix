### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/Jawiner/autoclix
   ```
2. Go to main folder
   ```sh
   cd autoclix
   ```
3. Run with python
   ```sh
   python autoclix.py
   ```
4. Enter ``Y`` if you want to install modules
